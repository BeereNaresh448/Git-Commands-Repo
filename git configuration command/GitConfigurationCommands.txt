Git Configuration Commands:
===========================
Git configuration can be done in Local/Project, Global/UserAccount and 
System Level (Git Software Installation)


Project/Local Level config Commands:
------------------------------------
To display the local config list: $git config --local --list

Display username :    $git config --local user.name
Display useremail:    $git config --local user.email

To Set New username :  $git config --local user.name newusername
To Set New useremail:  $git config --local user.name newuseremail

To remove username  :  $git config --local --unset user.name
To remove useremail :  $git config --local --unset user.email

To remove entire user section at local level config:

$git config --local --remove-section user
